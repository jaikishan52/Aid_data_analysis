General

Network graph visualization describes the relationship between donors and recipients based on the amount of money donated by the donor to the recipient and you can also observe how many recipients each donor has donated. For the network graph, the marks are going to be points and lines. Where points indicate the nodes and lines denote the links in the graph. Coming to channels we have area, shape, and color hue. The area indicates the amount of money donated or received whereas shape and color hue are used to categorize the donor and recipient

  • Color : Here we will be using categorical colormaps because our main aim in the stacked bar chart was to give unique colors for each purpose of donation so that they can be uniquely identified by the user just by looking at the graph. We have used discrete binning for the attribute (coalesced_purpose_name) to uniquely identify them. In our visualization, We should have perceptually distinct colors which will avoid confusion among the categories.
  • Transformation : created a calculated field by combining (donors, and recipients) which will help in forming the links between the nodes. So in creating the stacked bar chart, we ended up using the sum of the total amount donated for the bar size, the coalesced_purpose_name is used to categorize the following donation for which purpose it was named where it will be dividing the bar and its multiple segments with unique colors if the donation is made for multiple purposes by donor to the recipient. The year attribute has been added so that the user can see the following donations made by the user during a particular year.
  • Animation or Interaction : We have added filters for year and donor because we want the user to be able to get the following information, how much amount is donated by this donor to all the recipients and for which purpose the donation was made. And for the filters, we have multiple options in Tableau where we can add them in the form of single value (list), single value (dropdown), single value (dropdown), single value (slider), multiple values (list), multiple values(drop down), multiple values (custom list).
