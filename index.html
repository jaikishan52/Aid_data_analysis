Abstract

This topic gives detailed information about the Financial Aid Data Transactions between donors and recipients all over the world between 1991 to 2010. This data covers the information about the number of donors, recipients, amount donated in dollars and also it gives the purpose for which the amount donated. This topic mainly focuses on how much the amount has been donated by each country and the amount it is donated on what purpose to which country. This methodology involves data cleaning and data collection to enhance more reliability. The main aim of this report is to show the understanding of the International Aid data distribution showing trends and correlations which will be helpful for future donation strategies.

Introduction

Aid Data is an important policy tool for supporting economically developing countries. Between 1991 to 2010 most of the countries have donated lots of amounts based on different purposes for many developing countries(Recipients). This shows the data about the financial aid distribution globally. This dataset is rich in both categorical and quantitative data which gives us a unique opportunity to understand the complex details of global economic policies and the aid data. This project aims to make use of the data for showing the interactive visualizations by focussing on the years between 1991 to 2010. By creating interactive visualizations the team not only aims to clarify the distribution and impact of global financial aid but also seeks to inform policy negotiations and decision-making processes. This whole endeavor is supported by various tools, but we are choosing tableau and python which enables us to present complex data in an accessible manner. The visualizations we are using are highly interactive which allows users to explore the data in depth. These enhance the user experience significantly when compared to online static charts. For this Project we are planning to show two to three visualizations: Network Graph, Stacked Bar Chart, Sunburst Chart where each visualization shows the different information about the Aid Data. In Summary “Distribution of Global Financial Aid over the years” is a good project which leverages data visualization to visualize the complex data to the field of global data analysis, Local audience etc.,

Implementation

Visualization Tool and Programming language used: The visualization tools we used are Tableau and the Programming language is Python. For the Network graph and Stacked bar chart we used Tableau. For the Sunburst chart we have written our program in python language. We used a Jupyter Notebook to write our code in python. We used python libraries like plotly, matplotlib in our code. Plotly.express is for creating an interactive sunburst chart based on the year and color which we selected in the dropdown menus. Software Architecture: We used Jupyter Notebook to create a Sunburst Chart and inside it we used a pip installer for installing the required libraries. Also we used Tableau for creating Network Graph and Stacked Bar Chart. We stored our data dataset locally in excel file format for retrieving the data. In order to retain all the generated intermittent files we used pandas in python, by this the data transformation has been done.
